Coding

デザインと同じく 2011年から主にマークアップを書いてます。CSS はコードの複雑化による崩壊を防ぐため、SMACSS などを使った設計を心がけています。JavaScript に関してはいつまでたっても半人前以下の状態から抜け出せていませんが、Web 開発に携わる以上はできるだけ近いうちに習得したいと考えています。

あまり多くはありませんが、Git や Grunt などの開発に関わるツールも使ったり、試したりしています。

About

自己紹介

2011年から Webデザイナーとして活動しています。普段はサービスの企画段階から加わってデザインを作り、それをコーディングに落とすところまでやっています。

デザインに関しては人の目を惹くグラフィックを作ることも重要だと考えていますが、パソコンや携帯のように利用者が何かを操作する媒体では、順路がわかりやすい設計や機能の快適な使い心地を考えることがより重要だと思っています。

扱うスキルや使用しているサービス、趣味の麻雀などについてブログを書いているのでよければ覗いてみてください。おそらく僕が Web で一番活動している場がここになります。
